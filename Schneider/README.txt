Traffic packages exemples of schneider M200 is in folder Schneider_M200.
Traffic packages exemples of schneider_M221 is in folder Schneider_M221.
UMAS.lua is a wireshark plugin to resolve UMAS protocol. 
HOW TO USE UMAS.lua:
Paste it in folder plugins in wireshark installation directory and start wireshark. Wireshark will resolve UMAS protocol automatically.
